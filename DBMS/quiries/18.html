Find out the record of student whose name is of length 4?

~We can find the The record of student whose name is of length 4 using LIKE command.LIKE command is used in a WHERE clause to search for a specified pattern in a column.
You can use two wildcards with LIKE:
% - Represents zero, one, or multiple characters
_ - Represents a single character (MS Access uses a question mark (?) instead)

Syntax:-

select Name from hat where name like "____";
