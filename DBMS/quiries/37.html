How can we use CHECK constraint in the table?

~Check constraint is validation or a rule which is applied on the column of a table.
~When we apply check constraint to any column, it checks for specific values while inserting any record.
~Check constraint can be created on a table level as well as column level.
~Check constraints can be applied only to a single column, but there can be multiple check constraints in a single table.

Syntax:-

Column level:-CREATE TABLE TableName(ColumnName1 Datatype(size), ColumnName2 Datatype(size), ColumnName3 Datatype(size) Constraint ConstraintName CHECK(ColumnName CONDITION Value),…, ColumnNameN Datatype(size));
Table level:-CREATE TABLE TableName(ColumnName1 Datatype(size), ColumnName2 Datatype(size), ColumnName3 Datatype(size),…, ColumnNameN Datatype(size), Constraint ConstraintName CHECK(ColumnName CONDITION Value));
