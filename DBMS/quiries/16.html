CONDITONAL RETRIEVAL OF ROWS (WHERE clause)

                CONDITONAL RETRIEVAL of rows/records from a database table is done by specifying the condition in the WHERE clause of the SELECT statement.

WHERE Clasue:-

          The WHERE Clause is used when you want to retrieve specific information from a table excluding other irrelevant data. For example, when you want to see the information about employees belonging to department number 20 only, then you don't need the information about other employees. Retrieving information about all the employees would increase the processing time for the query.
           So, SQL offers a feature called WHERE clause, which we can use to restrict the data that is retrieved. The condition you provide in the WHERE clause filters the rows retrieved from the table and gives you only those rows which you expected to see. WHERE clause can be used along with SELECT, DELETE and UPDATE statements.

~The WHERE clause is used to filter records.
~It is used to extract only those records that fulfill a specified condition.

Syntax:-

SELECT column1, column2, ... FROM table_name WHERE condition;
